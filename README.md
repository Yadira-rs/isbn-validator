#  Validador de ISBN (ISBN Validator)

Este proyecto implementa funciones para validar códigos ISBN-10 e ISBN-13, junto con pruebas unitarias, pruebas de propiedades y automatización con GitHub Actions.

---

##  Objetivos

- Validar correctamente ISBN-10 e ISBN-13.
- Aplicar pruebas unitarias y property-based.
- Automatizar pruebas con CI/CD.
- Documentar trazabilidad y cobertura.
